Python超简单截取中文字符串

Posted on 三 11 七月 2012
by Wood
in Python.

web应用难免会截取字符串的需求,Python中截取英文很容易:

>>> s = 'abce'
>>> s[0:3]
'abc'

但是截取utf-8的中文机会截取一半导致一些不是乱码的乱码.其实utf8截取很简单,这里记下来作为备忘

#-*- coding:utf8 -*-
s = u'中文截取'
s.decode('utf8')[0:3].encode('utf8')
# 结果u'中文截取

def p():

def funcfactor(func):
    print 'calling function named', func.__name__
    func()
    print 'end'

funcfactor(p)
# 输出为:
# calling function named p
# Hello,world
# end

in Python.

easy_install beaker
